# IAM - 身份识别与访问管理(Identity and Access Management)

该项目旨在用一个完整的项目，来展示如何用go语言做企业级的后端服务开发。项目涵盖开发的各个流程和比较重要的功能点实现方式。
具体涵盖如下流程：设计 -> 开发环境配置 -> 开发 -> 单元测试 -> 功能测试 -> 部署
